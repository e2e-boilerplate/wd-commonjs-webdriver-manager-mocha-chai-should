# WD Boilerplate
    
WD end-to-end test automation boilerplate, Using [Commonjs](https://requirejs.org/docs/commonjs.html), [Webdriver-Manager](https://github.com/angular/webdriver-manager#readme), [Mocha](https://mochajs.org), [Chai](https://www.chaijs.com) and [Commonjs](https://www.chaijs.com/api/bdd/).
    
## Getting Started
    
1. git clone git@github.com:e2e-boilerplate/wd-commonjs-webdriver-manager-mocha-chai-should.git
    
2. cd wd-commonjs-webdriver-manager-mocha-chai-should
    
3. npm install
    
4. npm run update:webdriver
    
5. npm run test
        
    
For more boilerplate click [here](https://github.com/e2e-boilerplate/utils/blob/master/docs/implemented.md)
